# SmartText Summarizer

AI-powered text summarization web application built using Django and Hugging Face Transformers.

## Features
- Abstractive text summarization (BART, T5)
- Word count & compression ratio
- Language detection
- Clean Bootstrap UI

## Technologies Used
- Python
- Django
- Transformers (BART, T5)
- Bootstrap

## How to Run
```bash
pip install -r requirements.txt
python manage.py runserver
